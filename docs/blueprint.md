# **App Name**: Phage AI Research Assistant

## Core Features:

- Data Importance Summarization: Analyzes ML outputs and uploaded notes to identify and summarize the most biologically important signals.
- Continue ML Research: Extends existing ML analysis with deeper reasoning and insights, building on prior summaries without reprocessing raw genome data, functioning as a 'tool'.
- Chatbot Interface: Provides a chatbot interface to answer domain-related questions using research notes and ML summaries, while maintaining conversation history and supporting navigation of previous research sessions.
- Report Generation: Generates comprehensive reports aggregating summaries, interpretations, and insights in neutral, academic language suitable for PDF export.
- PDF research note upload: Ability to upload and analyze PDF based research notes, while ignoring non-relevant data.
- User greeting: Friendly, helpful greeting upon user login

## Style Guidelines:

- Primary color: Soft purple (#A78BFA) to evoke scientific creativity.
- Background color: Light gray (#F5F3FF) to keep focus on content.
- Accent color: Pale lavender (#D8B4FE) to provide contrast and call attention to actions.
- Headline font: 'Space Grotesk', a sans-serif with a computerized feel, paired with 'Inter' for body text. 
- Code font: 'Source Code Pro' for displaying any code or structured ML outputs.
- Simple, clean icons to represent different types of data and actions within the research assistant, with a professional yet approachable aesthetic.
- Subtle animations for the assistant character to convey attention cues and feedback, ensuring animations remain non-intrusive and scientifically professional.
